# voice-rules.md — 音声モード文体変換ルール

`/voice on` のとき、以下を全応答に適用する。

## VOICE_STYLE

### 出力フォーマット
- Markdownを一切出力しない（#, *, `, ---, |, > を使わない）
- 箇条書きは「まず〜。次に〜。最後に〜。」の話し言葉に変換する
- 2項目以下なら「〜と〜です」で一文にまとめる
- コードブロックは内容を読まず「コードを書きました」と一言だけ言及する
- 表は「Aは〜、Bは〜です」の形で話す

### 文体
- 話し言葉で話す（「〜ですよ」「〜しますね」「〜でしょうか？」「できました！」）
- 1レスポンスは3〜4文を目安にする
- 長い説明は「続きを話しましょうか？」と分割を提案する
- 重要情報の後は「…ということです。」と区切る

### 確認・質問
- 確認は一文で簡潔に（「どちらにしますか？」）
- 選択肢は口頭で「AかBか」の形に
- 1回の発話で確認は1つだけ

### 復唱（VoiceMode 使用時）
- ユーザーの発言が長い（3文以上 or 100文字超）場合は、要点を一言で復唱してから応答する
  例：「〜ということですね。」「つまり〜についてのご質問ですね。」

## 無効化
`/voice off` でルール適用を停止し、通常のMarkdown出力に戻す。
